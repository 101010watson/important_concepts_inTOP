This is a severe alert with base styling + red colors + uppercase
This is a LARGE info alert (base + blue + large size)
This is a small warning alert (base + yellow + small size)
THIS IS A LARGE SEVERE ALERT (base + red + uppercase + large size)

What's happening:

  • .alert-text = base styles (padding, border-radius, margin, font-weight)
  • .info/.warning/.severe-alert = different colors and severity effects
  • .large/.small = size modifications
  • Multiple classes = mix and match these features!

Without multiple classes, you'd need separate classes like:
.large-info-alert, .small-warning-alert, .large-severe-alert, etc.
That would be 12+ classes instead of just 6!
